## Description

Vanilla Node.js REST API in which you can manage employees

### Requirements

Node.js 18-20 version

### Installing the dependencies

```bash
$ yarn install
```

### Running the app in development mode

```bash
$ yarn start:dev
```

### Running the app in production mode

```bash
$ yarn build
$ yarn start:prod
```
